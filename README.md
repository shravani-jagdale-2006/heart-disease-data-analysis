# ❤️ Heart Disease Data Analysis

## 📌 Project Overview
This project focuses on analyzing a heart disease dataset to identify key risk factors using Exploratory Data Analysis (EDA).

## 📊 Key Steps
- Data Cleaning (handled missing values)
- Outlier Detection
- Data Visualization
- Feature Analysis

## 🔍 Key Insights
- Cholesterol and blood pressure are major risk factors
- Smoking increases heart disease probability
- Exercise reduces risk
- Family history plays a crucial role

## 🛠 Tools Used
- Python
- Pandas
- NumPy
- Matplotlib
- Seaborn

## 📁 Dataset
- Contains 10,000 rows and 21 columns
